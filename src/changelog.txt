Что изменилось по сравнению с оригинальной версией:
---Исходящие----
1) Для отправки сообщения с переводом на нужный язык(по дефолту английский) перед сообщением нужно ввести точку
пишем: .что тут происходит?
отправляется: what's going on here?
получаем для себя в чате тулбокса: Orig(ru-en): что тут происходит?
Без точки вначале отправится оригинальное сообщение и запроса на перевод в гугл api не будет, поэтому в настройках можно не выключать sendMode даже если он пока не нужен.
----------------

---Входящие----
2) (при условии targetLang = "ru") Нет странного перевода с "русского" на русский, т.к. теперь игнорируется кириллица
3) Не переводит сообщения, в которых всего одна буква(без учета знаков !:@.,? /\ и тд и цифр). Например :D, o/, r, r!, r?, r.&*!@#123123123 не будет переводится
4) Добавлен словарик для сообщений, которые не будут переводится. Его можно найти в src/exception-words.js и редактировать под себя - добавлять/удалять позиции.
Как работает: приходит сообщение ' CD 30! '
из него убираются знаки и цифры -> ' CD  '
убираются пробелы по бокам -> 'CD' (между буквами пробелы не убираются. 'hello world' останется 'hello world', так что в словарик можно добавлять предложения)
приводится к нижнему регистру -> 'cd'
значение ищется в словарике -> есть совпадение - перевода не будет
Для чего нужен: не засорять чат кривым переводом сокращений слов по типу ty brb cd xd ready и пр.
5) Переведенное сообщение посылается в таком виде (исходный язык-целевой язык): сообщение
Например, пишет Nishiyo на немецком:
Nishiyo: hallo, lass uns spielen
Nishiyo: (de-ru): привет, давай поиграем
--------------
